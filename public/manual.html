InfoTech - Manual de Usuario

Introducción

Bienvenido a la documentación del sistema de gestión de pedidos para InfoTech.

Este sistema permite gestionar clientes, libros, presupuestos y pedidos de impresión, además de ofrecer una vista pública para que los clientes consulten el estado de sus encargos.

Tip: Podés volver a este manual en cualquier momento haciendo clic en el botón Ayuda ubicado en la esquina superior derecha del panel.

Vista Pública (Rastreo)

Accesible desde la página principal (index.html).

  • Funcionalidad: Permite a los clientes ingresar su Código de Seguimiento (ej: 12345) para ver el estado actual de su pedido y el saldo pendiente.
  • Estados posibles: En cola de impresión, Imprimiendo / Armando, Listo para retirar, Entregado.

Panel de Administración

Accesible desde admin.html. Se divide en tres pestañas principales:

1. Gestor de Pedidos

Es la pantalla principal, rediseñada para agilizar la atención al público.

Buscador Visual (Izquierda)
  • Tarjetas: Los libros aparecen con su portada y precio grande.
  • Sugerencias: Si el buscador está vacío, te muestra los Más Vendidos de los últimos 30 días.
  • Info Detallada: Hacé clic en el botón de cada libro para ver estadísticas de ventas y stock.
  • Zoom: Clic en la foto pequeña para verla en grande.
Carrito y Acciones (Derecha)
  • Cliente Rápido: Usá el botón para registrar un cliente nuevo sin salir de la pantalla.
  • Agrupación: Si agregás el mismo libro varias veces, se agrupa en una sola línea (ej: Matemática 1 (x2)).
  • Atajos de Teclado:
    • F1 Confirmar Pedido: Guarda y genera el mensaje de WhatsApp.
    • F2 Presupuestar: Genera un presupuesto rápido para enviar.
  • Validación de Seña: El sistema te avisa si intentás cargar una seña mayor al total.

2. Gestión Inteligente (ABM)

El sistema aprende de lo que cargás para facilitar futuras cargas.

  • Autocompletado: Al cargar un libro o cliente, si escribís una Editorial o Colegio nuevo, el sistema lo guarda. La próxima vez, aparecerá en la lista de sugerencias.
  • Títulos de Libros: Al cargar un libro nuevo, te sugiere títulos existentes para que ahorres tiempo (ej: escribís "Matemática" y te sugiere "Matemática 1", solo cambiás el número).
  • Formato Automático: Todos los nombres y títulos se guardan automáticamente con la primera letra en mayúscula.

4. Lista de Pedidos

  • Visión General: Tabla con todos los pedidos históricos ordenados por fecha.
  • Estados y Colores:
    • En cola Pedido recién ingresado.
    • Imprimiendo En proceso de producción.
    • Encuadernando En proceso de anillado/terminación.
    • Listo para retirar Terminado, esperando al cliente.
    • Entregado Finalizado.
  • Progreso: Columna que muestra cuántos libros están terminados del total (ej: 2/3). Si el pedido está completo, la fila se pone verde.
  • Acciones:
    • Ver Detalle: Muestra qué libros pidió el cliente y los montos.
    • Avisar Cliente: Si el pedido está listo, aparece un botón verde para copiar el mensaje de aviso a WhatsApp.
    • Cambio de Estado: Se puede cambiar el estado general del pedido desde el botón desplegable.

5. Cola de Producción

Herramienta para organizar el trabajo del taller, agrupando pedidos por libro.

  • Agrupación Inteligente: Los libros se agrupan por Título. Si 10 clientes pidieron "Matemática 1", verás una sola entrada con el total de copias a realizar.
  • Filtros:
    • Por Título: Escribí en el buscador para encontrar un libro específico.
    • Por Estado: Usá el botón "Filtrar Estados" para ver solo lo que está "En cola" o "Encuadernando", por ejemplo. Podés seleccionar varios a la vez.
  • Gestión de Estados:
    • Individual: Cada libro tiene un selector para cambiar su estado (En cola -> Imprimiendo -> Encuadernando -> Terminado).
    • Masiva: Dentro de cada grupo, podés seleccionar varios libros (o todos) y cambiarlos de estado con un solo clic usando el selector de "Acción para seleccionados".
  • Indicadores Visuales: Barras de progreso por libro y contadores de stock en tiempo real.

6. Configuración

Permite personalizar los datos clave del negocio sin necesidad de programación.

  • Datos de Transferencia: Acá podés definir el Alias, Banco y Titular.
  • Automatización: Estos datos se insertan automáticamente en el mensaje de WhatsApp cuando generás un presupuesto (F2).
  • Seguridad: El panel te muestra un recuadro con la "Configuración Actual Activa" para que siempre sepas qué datos se están enviando a los clientes.
