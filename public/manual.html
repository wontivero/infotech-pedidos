InfoTech - Manual de Usuario

Introducción

Bienvenido a la documentación del sistema de gestión de pedidos para InfoTech.

Este sistema permite gestionar clientes, libros, presupuestos y pedidos de impresión, además de ofrecer una vista pública para que los clientes consulten el estado de sus encargos.

Vista Pública (Rastreo)

Accesible desde la página principal (index.html).

  • Funcionalidad: Permite a los clientes ingresar su Código de Seguimiento (ej: 12345) para ver el estado actual de su pedido y el saldo pendiente.
  • Estados posibles: En cola de impresión, Imprimiendo / Armando, Listo para retirar, Entregado.

Panel de Administración

Accesible desde admin.html. Se divide en tres pestañas principales:

1. Gestor de Pedidos

Es la pantalla principal de trabajo para la toma de pedidos.

Buscadores Inteligentes
  • Clientes: Buscá por nombre o teléfono. El sistema filtra en tiempo real.
  • Libros: Buscá por título, editorial o precio. Podés navegar con las flechas del teclado y presionar Enter para agregar al carrito rápidamente.
Carrito y Totales
  • Los libros agregados aparecen en una tabla con su miniatura (hacé click en la imagen para agrandarla).
  • Podés eliminar ítems con el botón de basura.
  • Cálculo de Seña: Al ingresar el monto en "Seña Pagada", el "Saldo Pendiente" se actualiza automáticamente.
Presupuestos
  • Generar Presupuesto: Guarda los ítems actuales bajo un código que empieza con P (ej: P12345). Copia automáticamente un mensaje formateado para WhatsApp al portapapeles.
  • Cargar Presupuesto: Ingresá el código (ej: P12345) en el campo superior y clickeá "Cargar". Esto recupera el cliente, los libros y los montos para finalizar la venta rápidamente.
Generar Pedido
  • Guarda el pedido definitivo en la base de datos.
  • Genera un Código de Seguimiento numérico (ej: 54821).
  • Copia al portapapeles un mensaje de confirmación listo para enviar al cliente con el link de seguimiento.

2. ABM Libros

  • Permite cargar nuevos libros al catálogo.
  • Imagen: Se sube a la nube y se muestra una miniatura en las listas.
  • Campos: Título, Páginas, Editorial, Precio.

3. ABM Clientes

  • Registro rápido de clientes con Nombre, Teléfono y Colegio.
  • Estos clientes aparecen luego en el buscador del Gestor de Pedidos.
